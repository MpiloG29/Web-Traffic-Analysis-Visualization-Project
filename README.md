
#  Web Traffic Analysis & Visualization Project

##  Project Overview
This repository contains the complete code and documentation for our group research project on "Python for Visualization & Reporting." Using a week's worth of web traffic data, we demonstrate advanced visualization techniques and automated reporting pipelines.

##  Team Members & Contributions
- **Member 1 (Excel Specialist)**: Data modeling, pivot tables, Excel dashboard, scenario analysis
- **Member 2 (Python Visualizations )**: Matplotlib trend plots, Seaborn heatmaps, Plotly interactive charts
- **Member 3 (Reporting Expert)**: Automated PDF/HTML report generation pipeline
- **Member 4 (Presentation Lead)**: PowerPoint design, insights narrative, final delivery

##  Technologies Used
- **Python 3.9+**: Core programming language
- **Pandas**: Data manipulation and analysis
- **Matplotlib**: Static trend visualizations
- **Seaborn**: Statistical heatmaps and correlations
- **Plotly**: Interactive dashboards and charts
- **ReportLab/FPDF2**: PDF report generation
- **Jinja2**: HTML templating for reports
- **Excel**: Data modeling and dashboard

##  Key Visualizations Created (Member 2)

### Static Visualizations (Matplotlib)
- Daily traffic trends with confidence intervals
- Multi-line plots showing traffic by source
- Page-wise performance comparisons


### Statistical Visualizations (Seaborn)
- Bounce rate correlation heatmaps
- Page x Source performance matrices
- Visitor behavior patterns
  

### Interactive Visualizations (Plotly)
- Dynamic bar charts with hover information
- Interactive pie/donut charts
- Scatter plots with trendlines
- Multi-plot dashboard with filters
